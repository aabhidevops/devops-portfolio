# 01 - Linux & Bash Scripting

## Project: System Info Script

A bash script that displays real-time system information including CPU, memory, disk usage, and OS details.

## How to Run
```bash
chmod +x system-info.sh
./system-info.sh
```

## What it shows
- Hostname, OS, Kernel version
- CPU model
- Memory usage
- Disk usage

## Skills Demonstrated
- Bash scripting
- Linux system commands
- Text processing with awk
